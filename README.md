# ML for Thermal Engineers

This repository documents a structured learning journey into **Machine Learning and AI from the perspective of a thermal systems engineer**.

The goal is to bridge concepts from:

- Automotive HVAC systems
- Vehicle thermal management
- System identification and calibration
- Data-driven modeling

with modern **Machine Learning techniques**.

Rather than focusing on generic datasets, this repository explores ML concepts using examples relevant to **thermal engineering and automotive systems**.

---

## Motivation

Thermal engineers often work with relationships such as:

- Ambient temperature vs compressor power
- Cabin conditions vs comfort perception
- Battery temperature vs cooling demand

Traditionally these relationships are modeled using:

- empirical equations
- calibration maps
- system identification techniques

Machine Learning provides an alternative approach where models **learn these relationships directly from data**.

This repository explores that transition.
